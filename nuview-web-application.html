Nuview Web Application

www.nulogx.com

Nulogx is a leader in transportation management solutions. NuView introduces the ability to track truck drivers while they are en route, predict their estimated arrival time at their next stop, and, if the driver is going to be late, send a message to the dispatcher so that the appointment can be rescheduled.

For this undertaking I was in charge of wireframing the Android application, UI and UX for the web application, as well as helping outline the the design for the new system.
